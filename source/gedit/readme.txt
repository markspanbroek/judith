Gedit syntax highlighting for *.judith files.

To install on Ubuntu:
- copy judith.lang to /usr/share/gtksourceview-3.0/language-specs
- copy judith.xml to /usr/share/mime/packages
- run update-mime-database /usr/share/mime

To install on OS X:
- copy judith.lang to /Applications/gedit.app/Contents/Resources/share/gtksourceview-3.0/language-specs
- copy judith.xml to /Applications/gedit.app/Contents/Resources/share/mime/packages
